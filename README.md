# Marketing Funnel Analysis

## Project Overview
This project analyzes customer journey stages and conversion funnel performance to identify drop-offs and business optimization opportunities.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Major customer drop-offs occurred during funnel stages
- Payment confirmation rates were significantly lower
- Desktop users generated higher traffic
- Funnel optimization can improve conversions
